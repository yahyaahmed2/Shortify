Shortify is a URL shortener built with Node.js, and MongoDB database, and utilizing JWT for authentication.

## Features 
- Shorten long URLs into compact links
- Redirect short links to original URLs
- Optional user authentication with JWT
- MongoDB backend with proper indexing for performance
- Simple RESTful API for integration
- Error handling and input validation

## Stack

- Node.js
- MongoDB
- Mongoose
- JWT
- Git & Github
- Express.js

## Future features and enhancements

- link management
- link analytics
- Redis caching for speed
- Rate limiting to prevent abuse
- deployment with CI/CD pipelines

inspired by Bitly, and TinyURL
